## Project Overview

**Pranam** is a full-stack e-commerce platform built using the **MERN stack**, featuring **separate Admin and User dashboards**. The application demonstrates modern web development practices including authentication, role-based access control, global state management, and RESTful API integration.

---

## Features

### Admin Dashboard
- Product management: **Create, Read, Update, Delete (CRUD)**
- Order management: Track and update orders
- User management: Control user roles and access

### User Dashboard
- Browse products by category
- Add/remove products to/from cart
- Place and track orders

### Authentication & Security
- **JWT-based authentication** for secure login
- **Role-based access control** for Admin and User operations

### State Management
- **Redux Toolkit** used for managing global state efficiently

### Backend Design
- **Node.js + Express** for server-side logic
- **MongoDB** for database storage
- **RESTful APIs** for frontend-backend communication
- Applied **OOP principles** for service and controller design

---

## Tech Stack
- **Frontend:** React, Redux Toolkit  
- **Backend:** Node.js, Express  
- **Database:** MongoDB  
- **Authentication:** JWT  
- **Deployment:** Render
